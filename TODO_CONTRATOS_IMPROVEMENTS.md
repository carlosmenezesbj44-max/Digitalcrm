# Melhorias no Menu de Contratos

## Melhorias Planejadas
1. **Expandir submenu de contratos**: Adicionar opções como "Editar Contrato", "Relatórios de Contratos", "Contratos Expirando".
2. **Melhorar filtros**: Adicionar filtro por data de criação, cliente específico, e valor.
3. **Adicionar paginação**: Para listas grandes de contratos.
4. **Ações em lote**: Permitir seleção múltipla de contratos para exclusão ou exportação.
5. **Melhorar responsividade**: Ajustar layout para dispositivos móveis.
6. **Notificações**: Alertas visuais para contratos expirando em breve.
7. **Modal de edição**: Permitir edição rápida de contratos no modal de detalhes.
